# Syslog2irc

Simple Syslog to IRC Gateway with message filtering

## Installation
 
1. `$ git clone https://github.com/xmorpheus/syslog2irc.git`
2. `$ cd syslog2irc`
3. `$ cd config`
4. `$ copy config-default.yml config.yml`
5. `$ copy blacklist-default.yml blacklist.yml`
6. `$ $EDITOR config.yml`

## Usage

`$ ./bin/syslog2irc`
